# Dan Koe 写作方法论 ✍️

> 基于 Dan Koe 的写作体系，指导你从选题到发布的完整写作流程。

## ✨ 功能特点

- **APAG 框架**：Attention → Perspective → Advantage → Gamify 完整写作结构
- **钩子公式**：4 种经过验证的开头写法
- **30 分钟写作清单**：快速产出高质量内容
- **一人利基策略**：找到独特定位

## ⚠️ 前置要求

无特殊前置要求。

## 📖 核心框架

### APAG 写作结构

| 阶段 | 名称 | 内容 |
|------|------|------|
| **A** | Attention | 钩子/大想法，用痛点开场 |
| **P** | Perspective | 树立敌人：描述大众的错误观点 |
| **A** | Advantage | 提出新视角/解决方案 |
| **G** | Gamify | 提供步骤/挑战，像游戏任务一样清晰 |

### 钩子公式

1. **大问题 + 负面后果**
2. **大想法 + 预期打破**
3. **If/Then 条件句**
4. **残酷真相**
