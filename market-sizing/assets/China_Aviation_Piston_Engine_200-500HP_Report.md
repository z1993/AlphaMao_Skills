# 市场规模分析报告：中国 200-500HP 航空活塞发动机

> **市场**: 中国通用航空活塞发动机 (200-500HP)  
> **地域范围**: 中国大陆  
> **时间范围**: 2024 (当前) - 2028 (预测)  
> **分析日期**: 2026-02-09  
> **分析师**: Claude (Market Sizing Skill)

---

## 执行摘要

| 指标 | 数值 | 说明 |
|------|------|------|
| **TAM** | ¥25.5 亿 | 2024年中国活塞发动机总市场规模 (基于 $354M) |
| **SAM** | ¥11.5 亿 | 200-500HP 细分市场 (约占 45%) |
| **SOM** | ¥1.7 亿 | 目标 15% 市占率 (保守估计) |
| **CAGR** | 8.0% | 预计未来 5 年复合增长率 |

**核心结论**: 
中国通航活塞发动机市场规模约 25.5 亿元人民币，其中 200-500HP 功率段是主力市场（覆盖 SR20, R44 等主流机型），规模约 11.5 亿元。受低空经济政策驱动，预计市场将保持 8% 左右的稳健增长。国产替代是主要机会点，但需面对 Lycoming 和 Continental 的强力竞争。

## 1. 市场定义与边界

### 1.1 市场定义
- **产品范围**: 200-500HP 功率段的航空活塞发动机。该功率段主要用于高性能单发飞机 (如 Cirrus SR20/22)、轻型双发飞机 (如 Diamond DA42/62) 及轻型直升机 (如 Robinson R44)。
- **目标客户**: 飞行培训学校 (主要需求)、私人飞行爱好者、通航运营企业、工业级无人机制造商。
- **排除范围**: 超轻型飞机用的低功率发动机 (<150HP, 如 Rotax 912系列大部分型号)、涡桨/涡扇发动机。

### 1.2 地域边界
- **中国大陆市场**，包含进口发动机在华销售及国产发动机销售。

### 1.3 时间范围
- **基准年份**: 2024年
- **预测期**: 2024-2028年

## 2. TAM 计算 (Total Addressable Market)

### 2.1 计算方法
采用 **Top-Down (自上而下)** 与 **Bottom-Up (自下而上)** 结合验证。

### 2.2 计算过程

**Top-Down 方法 (主要依据)**:
根据 IndexBox 及市场报告，中国活塞发动机市场 2024 年总规模约为 3.54 亿美元。
按 1:7.2 汇率计算：
```
TAM = $3.54 亿 × 7.2 ≈ 25.5 亿元人民币
```

**Bottom-Up 验证**:
- **机队规模**: 约 4000-5000 架活塞飞机 (截至 2021 年约 3751 台引擎，按年增 10% 估算)。
- **年更新/大修率**: 约 10% (TBO 2000小时，年飞 200-300小时) -> ~500 台/年。
- **新机交付**: 约 300-500 架/年。
- **年总需求**: ~800-1000 台。
- **均价**: ~40 万元/台 (6-8 万美元)。
```
TAM (验证) = 800 台 × 40 万 = 3.2 亿元
```
*注：Top-Down 数据 ($3.54亿) 远大于 Bottom-Up 估算的整机新机对应的引擎价值 (3.2亿)。差异可能源于 Top-Down 数据包含了 维修服务(MRO)、零部件 以及 军用/特殊用途 无人机市场。考虑到“航空发动机市场”通常包含全生命周期价值，本报告采纳 Top-Down 数据作为广义 TAM。*

### 2.3 结果
- **TAM**: **¥25.5 亿元** (广义市场)，**¥3.2-4.0 亿元** (仅新机+大修引擎销售)

## 3. SAM 计算 (Serviceable Addressable Market)

### 3.1 筛选维度

| 维度 | 筛选条件 | 占比估算 | 说明 |
|------|----------|----------|------|
| **功率段** | 200-500HP | 45% | 该段覆盖 R44 (Lycoming O-540, 260HP), SR20 (Continental IO-360, 200HP), SR22 (Continental IO-550, 310HP) 等最畅销机型。 |
| **应用场景** | 通用航空 | 100% | 聚焦主要通航应用 |

*注：Rotax 912 (100HP) 等低功率引擎在运动类飞机 (LSA) 中占比也不小，但单价较低；200-500HP 单价较高，价值量占比更高。*

### 3.2 计算
基于广义 TAM (包含服务与备件):
```
SAM = TAM × 200-500HP占比
    = 25.5 亿元 × 45%
    = 11.475 亿元 (约 11.5 亿元)
```

## 4. SOM 计算 (Serviceable Obtainable Market)

### 4.1 竞争格局与假设
市场高度垄断：
- **Lycoming**: 市占率 ~32% (2020数据)，R44 直升机的主力引擎。
- **Continental**: AVIC 收购，市占率高，SR20/22 的标配。
- **Austro Engine**: 钻石飞机 (Diamond) 自用，煤油活塞发动机。
- **国产机会**: 芜湖钻石、航瑞动力等正在崛起，尤其在无人机和教练机领域。

### 4.2 目标市占率假设
假设作为新进入者或国产替代厂商：
- **目标市占率**: **15%** (主要切入新增教练机市场及国产无人机市场)。
- **依据**: 国产化替代政策支持 + 价格优势。

### 4.3 计算
```
SOM = SAM × 目标市占率
    = 11.5 亿元 × 15%
    = 1.725 亿元 (约 1.7 亿元)
```

### 4.4 有效市场 (仅引擎销售) SOM
如果仅看引擎销售 (TAM=4亿)，则 SOM = 4亿 * 45% * 15% ≈ **2700 万元**。
*注：这表明仅卖引擎硬件市场较小，必须切入 MRO 服务或高价值无人机领域。*

## 6. 风险与局限性

1. **Top-Down 数据口径风险**: 报告数据可能包含非民用或非活塞部分，导致 TAM 偏高。建议以 Bottom-Up (3-4亿元引擎销售) 作为保守底线。
2. **适航取证壁垒**: 航空发动机取证周期长，SOM 获取速度可能低于预期。
3. **燃油替代风险**: 100LL 航空汽油供应受限，重油活塞 (Diesel) 和电动化是长期威胁。
