# 市场规模呈现策略指南

不同受众需要不同的呈现方式。本指南帮助你针对受众调整报告结构。

---

## For Investors (投资人)

**核心原则**: 可信度 > 精确度

### 推荐结构
1. **Executive Summary** — 一句话市场规模 + 关键增长逻辑
2. **Bottom-Up 计算** (主打) — 投资人最信任自下而上的推导
3. **Top-Down 验证** — 与行业报告交叉验证
4. **增长驱动力** — 为什么市场会持续增长
5. **竞争格局** — 市占率分布，证明还有空间
6. **假设清单** — 透明展示所有假设及来源

### 关键技巧
- **先给结论**: 投资人时间有限，第一页就给 TAM/SAM/SOM
- **保守优于激进**: 宁可低估后 upside surprise，也不要被质疑假设
- **使用范围而非单点**: "$80-120亿" 比 "$100亿" 更可信
- **链接到收入**: SOM → 你的 revenue target → 融资逻辑
- **展示增长率**: CAGR 比绝对值更重要

### 投资人常问的问题
- "为什么 SOM 是 X%？你凭什么拿到这个份额？"
- "Top-Down 和 Bottom-Up 差多少？"
- "市场增长的核心驱动力是什么？会持续吗？"

---

## For Strategy (内部战略)

**核心原则**: 可执行 > 可信度

### 推荐结构
1. **市场全景** — TAM 概览，但重点在 SAM/SOM
2. **细分市场排名** — 按机会大小排序
3. **进入策略** — 先打哪个细分？为什么？
4. **渗透时间表** — 第1年/第3年/第5年的渗透率目标
5. **资源需求** — 实现 SOM 需要多少投入

### 关键技巧
- **聚焦 SAM/SOM**: 战略更关心 "我们能做什么" 而非 "市场有多大"
- **按细分排优先级**: 不是所有细分都值得进入
- **关联 GTM**: 市场规模 → 目标客户画像 → 销售策略
- **标注竞争强度**: 每个细分市场的竞争程度
