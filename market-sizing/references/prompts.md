# 后续提示库 (Follow-up Prompts)

完成初步市场规模分析后，使用以下提示深化研究。

---

## 1. 细分市场深挖

**场景**: 初步分析显示某个细分市场特别有吸引力

**提示模板**:
```
深入分析 [细分市场名称] 这个细分市场：
- 该细分的具体规模和增速
- 主要玩家和市占率
- 进入壁垒和差异化机会
- 更新 Excel 中的 SAM/SOM 计算
```

**示例**:
```
深入分析中小企业 HR SaaS 这个细分市场，员工规模 50-200 人
```

---

## 2. 敏感性分析

**场景**: 某些假设不确定，需要了解其影响

**提示模板**:
```
对以下假设做敏感性分析：
- [假设1] 变动范围: [下限] ~ [上限]
- [假设2] 变动范围: [下限] ~ [上限]

生成敏感性矩阵，展示对 TAM/SAM/SOM 的影响
```

**示例**:
```
对以下假设做敏感性分析：
- 市场增长率：5% ~ 15%
- 目标渗透率：20% ~ 40%
生成敏感性矩阵
```

---

## 3. 竞品画像

**场景**: 需要了解竞争格局来估算 SOM

**提示模板**:
```
分析 [市场名称] 的竞争格局：
1. 列出前 5-10 名主要玩家
2. 估算各自市占率
3. 分析各家定位和差异化
4. 近期重要动态（融资、产品、并购）
```

**示例**:
```
分析中国 HR SaaS 市场的竞争格局，重点关注北森、薪人薪事、肯耐珂萨
```

---

## 4. 增长预测

**场景**: 需要多年预测而非单点估算

**提示模板**:
```
预测 [市场名称] 未来 [N] 年的规模：
- 计算复合增长率 (CAGR)
- 识别主要驱动因素和抑制因素
- 生成逐年预测表
- 更新 Excel 增加时间序列
```

**示例**:
```
预测中国 HR SaaS 市场 2024-2028 年的规模，并分析驱动 CAGR 的关键因素
```

---

## 5. 场景分析

**场景**: 存在重大不确定性，需要多情景规划

**提示模板**:
```
构建三种情景分析：
1. **乐观情景**: [描述假设]
2. **基准情景**: [描述假设]
3. **悲观情景**: [描述假设]

分别计算各情景下的 TAM/SAM/SOM
```

**示例**:
```
构建三种情景：
1. 乐观：政策利好 + 数字化加速
2. 基准：维持当前趋势
3. 悲观：经济下行 + 企业缩减 IT 支出
```

---

## 6. 区域细分

**场景**: 需要了解地域差异

**提示模板**:
```
按地域拆分 [市场名称]：
- 一线城市 vs 二三线城市
- 华东/华北/华南 区域占比
- 各区域增速差异
- 更新 SAM 计算
```

---

## 7. 验证与交叉检验

**场景**: 对结果存疑，需要多角度验证

**提示模板**:
```
用以下方法交叉验证当前测算：
1. 与 [对标公司] 的收入规模对比
2. 用 Bottom-Up 方法重新计算
3. 与 [第三方报告] 的数据对比
4. 列出差异并解释原因
```
