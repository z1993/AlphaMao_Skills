# 股票研究提示词库

本文件包含股票研究工作流中使用的所有提示词，分为两部分：
1. **数据提取提示词**（用于 NotebookLM）
2. **分析阶段提示词**（用于代理模型）

---

## 第一部分：数据提取提示词（NotebookLM）

这些提示词用于 `notebooklm ask` 命令，从导入的资料中提取结构化数据。

### 提取任务 1：业务构成数据

```
请从年报和券商研报中提取以下信息，以结构化表格形式输出：

1. 收入结构：按业务线和地区拆分的收入占比（过去3-5年）
   - 格式：表格，行为业务线/地区，列为年份，单元格为收入金额和占比

2. 利润构成：各业务线的利润贡献及利润率
   - 格式：表格，行为业务线，列为年份，包含利润金额、利润率

3. 主要产品/服务描述及市场地位
   - 格式：列表，每个产品/服务一段描述（100-200字）

请确保数据完整且准确，如果某些年份数据缺失，请标注"数据缺失"。
```

### 提取任务 2：财务 KPI 数据

```
请从财报中提取以下关键财务指标（过去3-5年），以表格形式输出，每年一列：

**收入与盈利能力：**
1. 总收入（Revenue）
2. 净利润（Net Income）
3. 毛利率（Gross Margin %）
4. 净利率（Net Margin %）

**资本回报率：**
5. ROE（净资产收益率）
6. ROA（总资产收益率）
7. ROIC（投入资本回报率）

**财务健康度：**
8. 资产负债率（Debt to Equity）
9. 流动比率（Current Ratio）
10. 速动比率（Quick Ratio）

**现金流：**
11. 自由现金流（Free Cash Flow）
12. 经营性现金流（Operating Cash Flow）

**费用结构：**
13. 研发费用占比（R&D as % of Revenue）
14. 销售费用占比（SG&A as % of Revenue）

请确保所有数据来源于官方财报，如果某些指标无法获取，请标注"数据缺失"。
```

### 提取任务 3：估值数据

```
请提取以下估值相关信息：

1. 历史估值倍数（过去3-5年）：
   - PE（市盈率）
   - PS（市销率）
   - PB（市净率）
   - EV/EBITDA
   格式：表格，行为指标，列为年份

2. 券商研报中提到的目标价和估值方法：
   - 格式：列表，每个券商一条，包含：券商名称、目标价、估值方法（如 DCF、PE 倍数法等）

3. 主要竞争对手名单（用于后续联网查询）：
   - 格式：列表，包含公司名称和股票代码

请确保数据准确，如果某些年份估值数据缺失，请标注"数据缺失"。
```

### 提取任务 4：市场观点汇总

```
请汇总分析师观点，以结构化形式输出：

1. 各券商的评级和目标价：
   - 格式：表格，列包含：券商名称、评级（买入/持有/卖出）、目标价、报告日期
   - 按目标价从高到低排序

2. 主流共识观点（3-5 条）：
   - 格式：列表，每条观点一段话（50-100字）
   - 提取多数分析师都认同的核心观点

请确保观点来源于券商研报，不要添加个人推测。
```

### 提取任务 5：管理层与分析师观点原文

```
请提取以下原始信息（供后续分析使用），以原文摘录形式输出：

1. 管理层在电话会中强调的未来增长驱动因素：
   - 格式：引用原文，每条一段，标注来源（哪次电话会、日期）
   - 重点提取管理层对未来战略、新产品、新市场的描述

2. 分析师关注的关键风险点：
   - 格式：引用原文，每条一段，标注来源（哪份研报、券商、日期）
   - 重点提取分析师明确指出的风险因素

3. 行业趋势和宏观因素影响的描述：
   - 格式：引用原文，每条一段，标注来源
   - 重点提取对行业周期、政策变化、技术趋势的分析

请确保所有内容为原文摘录，不要总结或改写。
```

### 提取任务 6：核心风险与下行推演

```
请从 10-K 年报的 "Item 1A. Risk Factors" 章节和近期研报中提取最具体的定量风险：

1. 核心风险因子 (Top 5):
   - 提取具体的、可量化的风险（如：单一客户贡献收入 40%，未决诉讼金额 5亿美元）。
   - 避免提取通用风险（如"宏观经济波动"）。

2. 悲观情景 (Bear Case) 线索:
   - 寻找分析师或管理层提到的"最坏情况"描述。
   - 提取可能导致股价暴跌的触发条件。

请以列表形式输出，每条风险必须包含具体数据或引用来源。
```

### 提取任务 7：前瞻指引与催化剂

```
请从最近一次 Earnings Call (业绩电话会) 和管理层讨论中提取前瞻信息：

1. 业绩指引 (Guidance):
   - 提取管理层对下一季度/全年的收入、利润、毛利率的具体预测数据。
   - 对比市场预期（如有提到）。

2. 近期催化剂 (Upcoming Catalysts):
   - 未来 6-12 个月的重大事件时间表（如：新品发布日、FDA审批日、投资者日）。
   - 每一个事件标注预期发生的时间。

请以结构化列表输出。
```

## 第二部分：分析阶段提示词（代理模型）

这些提示词用于代理模型基于提取的结构化数据进行深度分析。

### 阶段 1：业务构成与竞争力分析

```
请基于以下数据，详细分析该公司及其竞争对手的业务构成和竞争力：

**输入数据：**
[提取任务 1 的结构化数据]

**分析要求：**

1. **收入结构分析**（以表格形式呈现）：
   - 按业务线拆分：各业务线的收入占比及增长趋势（过去3-5年）
   - 按地区拆分：各地区的收入占比及增长趋势
   - 识别核心业务和增长引擎

2. **利润构成分析**：
   - 各业务线的利润贡献及利润率差异
   - 识别高利润率业务和低利润率业务
   - 分析利润率变化趋势及原因

3. **竞争优势/护城河分析**：
   - 相对竞争对手的核心竞争力（技术、品牌、规模、网络效应等）
   - 护城河的可持续性评估
   - 竞争格局变化趋势

**输出格式：**
- 使用表格呈现收入和利润数据
- 用 3-5 段文字分析竞争优势
- 明确指出核心业务和护城河类型
```

### 阶段 2：财务指标与健康度分析

```
请从专业财务分析师和审计师的视角，分析该公司的财务健康状况：

**输入数据：**
[提取任务 2 的结构化数据]

**分析要求：**

1. **核心 KPI 趋势分析**：
   - 识别 3-5 个最能代表该公司经营状况的关键指标
   - 展示过去 3-5 年的数据趋势（表格 + 趋势描述）
   - 分析趋势背后的驱动因素

2. **财务健康度评估（审计师视角）**：
   - 盈利能力：毛利率、净利率的变化及可持续性
   - 资本效率：ROE、ROA、ROIC 的表现及同行对比
   - 偿债能力：资产负债率、流动比率、速动比率的健康度
   - 现金流质量：自由现金流与净利润的匹配度
   - **红旗信号**：指出任何值得注意的异常信号或潜在风险

3. **费用结构分析**：
   - 研发费用和销售费用占比的合理性
   - 费用结构与业务模式的匹配度

**输出格式：**
- 使用表格呈现关键 KPI 趋势
- 用 3-5 段文字分析财务健康度
- 明确指出任何红旗信号
```

### 阶段 3：估值分析

```
请分析该公司当前的估值水平及其合理性：

**输入数据：**
- [提取任务 3 的历史估值数据]
- [联网获取的竞争对手实时估值数据]

**分析要求：**

1. **历史估值趋势**：
   - 过去 3-5 年的 PE、PS、PB、EV/EBITDA 等关键估值倍数变化
   - 识别估值高点和低点，分析原因
   - 当前估值在历史区间的位置

2. **同行对比**（表格形式）：
   - 与主要竞争对手的估值倍数对比
   - 分析估值溢价或折价的原因
   - 考虑增长率、盈利能力、风险等因素

3. **当前估值评估**：
   - 结合增长预期和行业周期，判断当前估值是：
     * 明显偏贵（溢价 >30%）
     * 略微偏贵（溢价 10-30%）
     * 合理区间（±10%）
     * 略微便宜（折价 10-30%）
     * 明显便宜（折价 >30%）
   - 给出估值判断的具体理由

4. **悲观情景折价 (Mandatory Discounting)**:
   - **必须结合 [提取任务 6] 中的风险因子**。
   - 假设核心风险（如大客户流失）发生，估值倍数应打多少折？
   - 计算 "Bear Case" (悲观情景) 的目标价。

**输出格式：**
- 使用表格呈现历史估值和同行对比
- 用 2-3 段文字给出估值评估结论
- 包含 Bull/Base/Bear 三种情景的目标价测算
- 明确标注当前估值水平（偏贵/合理/便宜）
```

### 阶段 4：市场共识与独特视角

```
请综合分析市场对该公司的看法，并挖掘独特洞察：

**输入数据：**
- [提取任务 4 的市场观点汇总]
- [所有已提取的结构化数据]

**分析要求：**

1. **分析师共识**：
   - 汇总主流券商的目标价和评级分布（表格形式）
   - 计算平均目标价和上涨空间
   - 总结 3-5 条主流共识观点

2. **独特视角（模型洞察）**：
   - **这部分不是从资料中摘录，而是基于所有数据进行深度推理**
   - 挖掘市场可能忽视或误判的关键因素，例如：
     * 被低估的业务线或资产
     * 尚未被充分定价的增长驱动因素
     * 市场过度关注的短期因素而忽视的长期价值
     * 与主流观点相悖但有数据支撑的判断
   - 每条独特洞察需要有明确的数据或逻辑支撑

3. **预期差分析**：
   - 市场预期 vs. 实际可能的情景
   - 哪些因素可能导致市场预期修正

**输出格式：**
- 使用表格呈现分析师共识
- 用 3-5 段文字阐述独特视角
- 明确标注哪些是独特洞察（非市场共识）
```

### 阶段 5：关键驱动因子与系统图谱

```
请识别影响该公司未来估值的关键驱动因素，并构建系统图谱：

**输入数据：**
- [提取任务 5 的管理层与分析师观点原文]
- [所有已提取的结构化数据]

**分析要求：**

1. **核心驱动因子识别（模型构建）**：
   - **这部分不是简单罗列资料中的观点，而是基于所有信息进行综合判断**
   - 识别 3-5 个最重要的价值驱动因素，例如：
     * 新产品/新市场的渗透率
     * 技术迭代周期
     * 行业政策变化
     * 成本结构优化
     * 市场份额变化
   - 每个驱动因子需要量化其潜在影响（如可能带来的收入/利润增长）

2. **业绩指引验证 (Guidance Check)**:
   - **必须结合 [提取任务 7] 的管理层指引**。
   - 驱动因子是否得到了管理层指引的支撑？
   - 如果管理层指引低于市场预期，必须将该因子标记为"负面/承压"。

2. **相互关系分析（因果链）**：
   - 这些驱动因子之间如何相互影响
   - 构建因果关系图（用文字描述因果链）
   - 识别关键杠杆点（哪个因子的变化会产生最大影响）

3. **情景分析**：
   - **乐观情景**：驱动因子全部向好，估值可能达到的水平
   - **基准情景**：驱动因子符合预期，估值的合理区间
   - **悲观情景**：驱动因子不及预期，估值可能的下行风险
   - 每个情景给出具体的估值倍数或目标价区间

**输出格式：**
- 列表形式呈现 3-5 个核心驱动因子及其量化影响
- 用文字描述因果关系图（如：A → B → C）
- 表格形式呈现三种情景下的估值预测
```

### 阶段 6：综合投资建议

```
基于以上所有分析，请给出针对以下投资偏好的综合建议：

**投资偏好：**
- 基本面扎实（财务健康、业务逻辑清晰）
- 估值合理（不追高，寻找合理估值区间）
- 有弹性（成长空间，未来有明确的增长驱动因素）

**输入数据：**
[阶段 1-5 的所有分析结果]

**分析要求：**

1. **投资论点**（1-2 句话）：
   - 用最简洁的语言概括核心投资逻辑
   - 例如："该公司在 XX 领域具有领先优势，当前估值处于历史低位，新产品周期即将启动"

2. **主要风险**（2-3 个）：
   - 列出最值得关注的风险点
   - 每个风险点评估其发生概率和潜在影响

3. **操作建议**：
   - 明确给出：建仓/加仓/持有/减仓/观望
   - 建议仓位（如：建议配置 5-10% 仓位）
   - 建议买入价格区间（如有）
   - 止损位和止盈位（如适用）

4. **关注催化剂**：
   - 未来 3-6 个月内，哪些事件可能触发股价变动
   - 例如：财报发布、新产品上市、政策落地等

5. **投资时间框架**：
   - 短期（3-6 个月）、中期（6-12 个月）、长期（1-3 年）的预期

**输出格式：**
- 投资论点：1-2 句话
- 主要风险：列表，每个风险 1-2 句话
- 操作建议：明确的行动指引
- 关注催化剂：列表，每个催化剂 1 句话
- 投资时间框架：简短描述

**免责声明：**
本建议仅供参考，不构成投资建议。投资者应根据自身情况独立决策。
```

## 使用说明

### 数据提取阶段

在 NotebookLM 中，对每个提取任务运行：
```bash
notebooklm ask "<提取提示词>" -n <notebook_id> --json
```

### 分析阶段

在代理侧，将提取的结构化数据作为输入，依次执行 6 个分析阶段的提示词。

### 提示词调整

如果用户有特定的投资偏好或分析需求，可以在阶段 6 的"投资偏好"部分进行调整。
